IBAN Validation — Everything You Need to Know

An IBAN (International Bank Account Number) is the international standard for identifying bank accounts (ISO 13616). Our free IBAN checker validates the format, country code, and MOD-97 checksum of any IBAN offline — no banking connection needed.

What Does IBAN Validation Check?

  • Country code: Valid ISO 3166-1 alpha-2 code from a SEPA or IBAN-supporting country
  • Length: Each country has a fixed IBAN length (16 for Belgium, 22 for Germany)
  • Check digits (MOD-97): Mathematical verification using ISO 7064 algorithm
  • BBAN format: Basic Bank Account Number matches country-specific pattern

IBAN Format by Country

Belgian IBANs are 16 characters (BE + 14). German IBANs are 22 characters (DE + 20). French IBANs are 27 characters (FR + 25). Within SEPA (31 countries), IBAN alone is sufficient for euro transfers — no BIC required since 2016.

Frequently Asked Questions

Does a valid IBAN mean the account exists?
No. IBAN validation confirms the format and checksum are correct, but cannot confirm the account is currently active. To verify an active account, you need bank-level verification (e.g., a small test transfer).
What is the IBAN format for Belgium?
Belgian IBANs are 16 characters: BE + 2 check digits + 3-digit bank code + 7-digit account number + 2-digit national check digits. Example: BE68 5390 0754 7034.
